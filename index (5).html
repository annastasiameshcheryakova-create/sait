Програма

Що ви вивчите

Курс побудований блоками від базових понять до створення повноцінного веб-додатку. Кожен модуль завершується практичним проєктом.

  • ✔ Розуміти, як працює веб та браузер
  • ✔ Верстати адаптивні сучасні інтерфейси
  • ✔ Писати чистий JavaScript без копіпаст
  • ✔ Створювати React-додатки
  • ✔ Писати сервер на Node.js та працювати з базою даних

  • Як працює Інтернет та браузер
  • HTML5: структура сторінки, семантичні теги
  • CSS3: селектори, властивості, моделі коробки
  • Flexbox та Grid, адаптивна верстка
  • Міні-проєкт: лендінг сторінка

  • Синтаксис, змінні, типи даних
  • Умови, цикли, функції
  • Робота з DOM та подіями
  • Асинхронність: проміси та async/await
  • Міні-проєкт: інтерактивний To-Do список

  • Компонентний підхід
  • Хуки useState та useEffect
  • Роутинг та робота з API
  • Проєкт: односторінковий застосунок (SPA)

  • Вступ до Node.js
  • Створення REST API
  • Робота з базами даних (MongoDB / PostgreSQL)
  • Авторизація та безпека
  • Фінальний проєкт: повноцінний веб-додаток
Аудиторія

Для кого цей курс

Курс підійде, якщо ви готові приділяти навчанню 6–8 годин на тиждень та хочете результат у вигляді реального коду, а не тільки теорії.

Для новачків

  • Ніколи не писали код, але давно хотіли спробувати
  • Переходите в ІТ із іншої сфери
  • Потрібна структурована база та підтримка

Для студентів

  • Хочете підтягнути практику до теорії з університету
  • Плануєте стажування або першу роботу в ІТ

Для спеціалістів інших напрямів

  • Маркетологи, дизайнери, менеджери продуктів
  • Хочете краще розуміти розробників
  • Потрібно вміти швидко перевіряти гіпотези в коді

Не підійде, якщо

  • Хочете результат без регулярної практики
  • Очікуєте, що курс сам знайде вам роботу
